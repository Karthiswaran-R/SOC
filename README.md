# Mini RISC-V SoC (System-on-Chip)

## Overview
This project implements a **Mini System-on-Chip (SoC)** built around a **RISC-V CPU core**, designed for FPGA-based prototyping and learning purposes.

It demonstrates:
- Hardware–Software Co-Design
- Bus-based communication
- Modular VLSI architecture

---

## Features
- Custom RISC-V CPU Core
- Instruction & Data Memory
- Shared Bus Architecture
- Multiple Peripheral Integration
- Interrupt Handling System
- FPGA Compatible Design

---

## SoC Components

### Core Blocks
- RISC-V CPU Core  
- Instruction ROM  
- Data RAM  
- Address Decoder  
- AHB Bus
- AHB to APB Bridge  

### Peripherals
- GPIO  
- UART  
- Timer / Counter  
- Watchdog Timer  
- SPI Master  
- I2C Master  
- Debug Interface  
- Interrupt Controller  

### System
- Clock & Reset Generator
